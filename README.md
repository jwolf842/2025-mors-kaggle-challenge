# 2025 MORS Kaggle Data Challenge
Stance detection for YouTube comments on U.S. Army videos.

**Competition:** [2025 MORS Data Challenge on Kaggle](https://www.kaggle.com/competitions/2025-mors-data-challenge)

## Files
- `2025 MORS Challenge Ensemble_DeBERTa_optimized2.py` - Main ensemble classifier
- `submission_ensemble_1.csv` - Generated predictions

## Approach
Ensemble combining sentiment analysis, zero-shot DeBERTa, and rule-based classification.
